IMPLEMENTACIÓN DEL SOFTWARE

¿Qué es?

La implementación de software es el proceso de poner en funcionamiento un sistema o aplicación de software en un entorno real, después de haber sido diseñado y desarrollado.

¿En qué se puede usar?

Se utiliza en diversas áreas, como la gestión empresarial, la educación, la salud, y más, para mejorar procesos, aumentar la eficiencia y facilitar la toma de decisiones.

Etapas de la implementación de software

Las etapas de la implementación de software son: Planificación, Configuración, Pruebas, Despliegue, Capacitación, y Soporte y Mantenimiento.

HARDWARE DE SERVIDORES

¿Qué es un Hardware?

Un servidor basado en hardware es una máquina física integrada en una red informática en la que,además del sistema operativo, funcionan uno o varios servidores basados ensoftware.

Rack

Es un servidor capaz de ofrecer una gran potencia de computación y capacidad de almacenamiento, con compatibilidad para varios procesadores, más memoria y más almacenamiento.

Blade

Son servidores muy finos y pequeños que se colocan en “bahías” y se conectan entre sí para realizar cálculos de gran potencia.

Arreglos de discos

RAID es un método de combinación de varios discos duros para formar una unidad lógica única en la que se almacenan los datos de forma redundante.

Enclousure

Un enclosure, en términos simples, es un gabinete o carcasa que alberga componentes electrónicos. Se presenta en una variedad de formas, tamaños y materiales, y su diseño puede adaptarse a necesidades específicas.

SISTEMAS OPERATIVOS DE SERVIDORES

Definición General

Un sistema operativo de servidor es un software que se encarga de gestionar los recursos para que puedan ser utilizados por diferentes aplicaciones y usuarios. Es esencial en la gestión de servidores web, bases de datos y grandes centros de datos.

Servidores Linux

Son servidores que ejecutan una variante del sistema operativo (SO) open source Linux. Están diseñados para gestionar las aplicaciones empresariales más exigentes, como los servicios web o las bases de datos.


Estos servidores de Linux se caracterizan por su desempeño el rendimiento que usted necesita que tenga su infraestructura, por la estabilidad ya que estos servidores so impulsados por tecnología de código abierto que admite capacidades de imagen, por servicios de seguridad ya que estos servidores ofrecen permisos mejorados que pueden optimizarse con fines de seguridad, y por su servicio de escalabilidad ya que Linux trabaja con tecnologías de la nube para ayudarle a escalar su empresa con mayor facilidad.

Servidores Windows

Es una plataforma para compilar una infraestructura de aplicaciones, redes y servicios web conectados del grupo de trabajo al centro de datos.


Windows Server está desarrollado en C++ y Asambler. Una de sus características más destacadas para equipos de trabajo es que es un sistema multiusuario. Por lo tanto, es un sistema que pueden utilizar todos los empleados de una determinada compañía, centralizando así la gestión y administración de archivos.


Las ventajas que ofrece Windows Server como Sistema Operativo son relevantes para favorecer el trabajo de los programadores y desarrolladores y, por tanto, mejorar los resultados corporativos.

WINDOWS SERVER

Diseñado para...

Windows Server está diseñado para gestionar, almacenar, enviar y procesar datos de manera continua, ofreciendo servicios a múltiples usuarios y dispositivos en una red. Las funciones comunes de un servidor Windows incluyen la administración de usuarios, la administración de políticas de seguridad, el alojamiento de sitios web, la gestión de archivos y aplicaciones, y el soporte para aplicaciones empresariales.

Requisitos Mínimos para Windows Server

  • Procesador de 1.4 GHz de 64 bits compatible con la arquitectura x64. Compatible con NX y DEP, CMPXCHG16b, LAHF/SAHF, PrefetchW y Second Level Address Translation (EPT o NPT).
  • Memoria RAM de 512 MB para Server Core o 2 GB para instalación con la opción Desktop Experience.
  • Espacio en Disco de al menos 32 GB. Se recomienda más espacio para instalaciones con el rol de servidor de archivos, para almacenamiento de datos y para actualizaciones futuras.
  • Tarjeta Ethernet con capacidad mínima de un gigabit por segundo (1 Gbps).

Licenciamientos

Licenciamiento por Núcleo (Core Licensing)

Windows Server Standard y Datacenter: Estas ediciones se licencian por el número de núcleos físicos en el servidor. Se requieren licencias para un mínimo de 16 núcleos por servidor y al menos 8 núcleos por procesador.

Licencias de Acceso de Cliente (CALs)

User CAL: Una licencia por cada usuario que accede al servidor, independientemente del número de dispositivos utilizados.

Device CAL: Una licencia por cada dispositivo que accede al servidor, independientemente del número de usuarios que lo utilizan.

Licenciamiento por Abonados y Servicios (Subscription Licensing)

Azure Hybrid Use Benefit: Permite a las empresas utilizar las licencias de Windows Server en entornos de nube de Azure a precios reducidos.

Licencias OEM (Original Equipment Manufacturer)

Compradas directamente con hardware nuevo, estas licencias son generalmente más económicas, pero están ligadas al hardware con el que se compraron.

Licencias para Proveedores de Servicios (SPLA)

Modelo de licencia de alquiler mensual para proveedores de servicios que deseen ofrecer Windows Server a sus clientes.

HOSTING Y DOMINIO

¿Qué es?

Hosting (Alojamiento Web): Es el servicio que proporciona espacio en un servidor para almacenar los archivos de un sitio web, haciéndolos accesibles a través de Internet.


Dominio: Es la dirección única que identifica a un sitio web en Internet (por ejemplo, www.ejemplo.com).

Tipos de Hosting

  • Hosting Compartido: Varios sitios web comparten los recursos de un mismo servidor.
  • Hosting VPS (Servidor Privado Virtual): Un servidor físico dividido en múltiples servidores virtuales, cada uno con sus propios recursos dedicados.
  • Hosting Dedicado: Un servidor completo dedicado a un solo sitio web.
  • Hosting en la Nube: Utiliza una red de servidores para alojar el sitio web, ofreciendo escalabilidad y redundancia.
  • Hosting Gestionado: Incluye servicios adicionales de administración y soporte, ideal para sitios web que usan CMS.

Configuración de Hosting y Dominio

Registro del Dominio

Proveedores: GoDaddy, Namecheap, Google Domains.


Proceso: Elegir un nombre de dominio disponible y registrar a través de un registrador.

Configuración del DNS

DNS (Sistema de Nombres de Dominio): Traduce nombres de dominio en direcciones IP.

Configuraciones clave

A Record: Apunta el dominio a una dirección IP específica.


CNAME Record: Apunta un subdominio a otro dominio.


MX Record: Especifica los servidores de correo para el dominio.

Configuración del Hosting

Subir Archivos del Sitio Web: Usar FTP/SFTP o el panel de control del proveedor.


Configurar Bases de Datos: Crear y gestionar bases de datos necesarias para el sitio web (usualmente MySQL).

FTP

¿Qué es?

El término "FTP" significa "Protocolo de Transferencia de Archivos" en inglés. Es un protocolo estándar de Internet que se utiliza para transferir archivos entre computadoras a través de una red. FTP permite la transferencia de archivos de una computadora a otra, ya sea en la misma red o a través de Internet, utilizando un cliente FTP para enviar archivos y un servidor FTP para recibirlos. Es una herramienta muy útil para compartir y gestionar archivos de forma remota.

Trasferencia de archivos FTP

La transferencia de archivos a través de FTP implica el uso de un cliente FTP para enviar archivos desde una computadora a un servidor FTP, donde los archivos se almacenan y se pueden compartir o gestionar. El cliente FTP puede ser un software dedicado o incluso un navegador web con capacidades integradas de FTP. Esta transferencia puede realizarse en redes locales o a través de Internet, lo que permite a las personas compartir y gestionar archivos de forma remota de manera eficiente.

DIRECCIÓN IP

¿Qué es?

Las direcciones IP son el identificador que permite el envío de información entre dispositivos en una red. Contienen información de la ubicación y brindan a los dispositivos acceso de comunicación. La dirección IP identifica de forma exclusiva cada uno de los dispositivos en Internet.
